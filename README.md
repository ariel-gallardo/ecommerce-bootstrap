	- Buscador
	- Logo de la empresa
	- Notificacion
	- Carrito

* Main
	- Productos
* Footer
	- Copyright
	- Anio
	- Redes Sociales

## Pasos
1. Instalar NodeJS.
2. Clonar proyecto - **git clone URL**.
4. Instalar dependencias - **npm install**
5. Configurar variables de entorno **.env**
6. Ejecutar proyecto - **npm start**
